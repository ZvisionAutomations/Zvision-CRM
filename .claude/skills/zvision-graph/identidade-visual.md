# Identidade Visual

A identidade do Zvision e War Room — um CRM que se parece com um centro de operacoes militares. Cada elemento visual reforça a ideia de precisao, urgencia e controle.

## Paleta de Cores

Os [[tokens-de-cor]] sao os seguintes:

- `accent-primary: #A2E635` — verde tatico — acoes, CTAs, dados criticos
- `accent-ai: #00D4FF` — azul eletrico — estados AI, atividade de agentes, insights
- `surface-page: #0A0A0A` — preto absoluto — background principal
- `surface-card: #111111` — superficie de cards
- `surface-elevated: #1A1A1A` — superficie elevada (modals, dropdowns)
- `border-default: rgba(255,255,255,0.06)` — bordas sutis
- `text-primary: #F0F0F0` — texto principal
- `text-secondary: rgba(240,240,240,0.5)` — texto secundario

## Tipografia

O sistema usa duas fontes com papeis claros — detalhes em [[tipografia]]:

- **Space Grotesk** — UI, titulos, display, navegacao
- **JetBrains Mono** — dados, labels, badges, codigos, metricas numericas

## Tom Tatico

O vocabulario nao e opcional — e parte da identidade. Usar os termos errados quebra a imersao:

| Errado | Correto |
|--------|---------|
| Cliente | Target |
| Notas | Intel |
| → | >> |
| Processar | Executar Missao |
| Analisar | Fazer Briefing |

## Grid de Fundo

Todas as paginas tem um [[grid-tatico]] de background: linhas em `rgba(162,230,53,0.03)` com espacamento de 40px. Sutil, mas cria a sensacao de interface militar.

## Principio Central

Cada componente, animacao e texto deve passar no teste: "isso parece um centro de operacoes de elite?" Se sim, esta correto.
