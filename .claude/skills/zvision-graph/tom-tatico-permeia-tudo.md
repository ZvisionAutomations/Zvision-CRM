# Tom Tatico Permeia Tudo

## A Claim

O vocabulario tatico do Zvision nao e decorativo — e funcional. Ele alinha o agente AI ao mindset do produto e alinha o usuario ao comportamento que o produto incentiva.

## Por que isso importa

Quando um agente recebe "analise o cliente", ele pensa como um CRM generico. Quando recebe "faca o briefing de inteligencia do target", ele opera no modo Zvision — com precisao, urgencia e contexto de operacoes de alto desempenho.

O mesmo vale para o usuario: um vendedor que usa "Target" em vez de "Cliente" inconscientemente adota uma postura diferente — mais estrategica, menos passiva.

## O Mapeamento Completo

| Conceito Generico | Zvision |
|---|---|
| Cliente / Lead | Target |
| Notas / Observacoes | Intel |
| Tarefa | Missao |
| Concluido | Mission Complete |
| Analisar | Fazer Briefing |
| Processar | Executar |
| → | >> |
| Dashboard | War Room |
| Automacao | Agente |
| Relatorio | Sitrep |

## Aplicacao Pratica

Todo texto visivel ao usuario — labels, tooltips, mensagens de sucesso/erro, placeholders — deve passar pelo filtro tatico. Consultar [[identidade-visual]] para o principio central de design.

Todo prompt enviado a APIs de AI deve usar o vocabulario Zvision para que o modelo responda no mesmo registro.
